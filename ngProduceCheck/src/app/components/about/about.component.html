Overview

This project is designed as a means to create and share recipes between users giving recipe books modern functionality such as: commentary on recipes from other users, user likes via reactions, and other various (now common) social media aspects. Additionally, it provides a link to local area stores to enable users to quickly see whether or not ingredients for the recipes are in stock.
